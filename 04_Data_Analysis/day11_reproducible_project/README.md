# Day 11 Reproducible Project Practice

This small practice project shows the Day 11 reproducible analysis structure.

## Structure

```text
data/raw/                 original input data, do not edit by hand
data/processed/           tables generated from raw data by scripts
results/figures/          figures generated by scripts
notebooks/                exploratory analysis notes
scripts/                  reproducible analysis code
```

## Reproduce Outputs

Run from this folder:

```powershell
python scripts/run_expression_pipeline.py
```

Expected outputs:

```text
data/processed/expression_summary.csv
results/figures/expression_raw_points.png
```

## Rule

Do not edit `data/raw/expression_samples.csv` during analysis. If the processed output needs to change, update the script and regenerate the output.
